## Tips
- Master modular arithmetic—it appears everywhere
- (a * b) % m = ((a % m) * (b % m)) % m
- For modular inverse: use Fermat's theorem when m is prime
- Prime factorization in O(√n)
- Sieve modifications for advanced problems
