# Arrays

Array-based problems covering manipulation, searching, sorting, and optimization techniques.

## Key Concepts Covered
- Array traversal and manipulation
- Two-pointer technique
- Sliding window
- Prefix sum / Range queries
- Kadane's algorithm (maximum subarray)
- Array rotation and reversal
- Majority element finding
- Interval problems

## Problems in this Directory
- `array_sorted/` - Check if array is sorted
- `array_majority_element/` - Find majority element (Boyer-Moore)
- `array_rainwatertrap/` - Rain water trapping problem
- `arrays_max_cont_subarray/` - Maximum contiguous subarray sum
- `arrays_merge_intervals/` - Merge overlapping intervals
- `equilibium/` - Equilibrium index
- `good_pair/` - Count good pairs
- `leader_in_array/` - Find leaders in array
- `max_sub_array/` - Maximum subarray problems
- `rotate_array/` - Array rotation techniques
- `special_index/` - Special index problems
- `sum_of_sub_arrays/` - Subarray sum calculations
- `sumofarray/` - Basic array sum
- And many more...

## Tips
- Always consider edge cases: empty arrays, single elements
- Think about time/space tradeoffs
- Consider sorting when order doesn't matter
- Use prefix sums for range queries
